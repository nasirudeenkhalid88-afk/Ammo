# Ammo - AI Animation Generator

## Features
- Character customization
- Voice customization
- Mock generation without API keys

## Installation Instructions
To install, run:
```
npm install
```

## Usage Guide
To start the application, use the following command:
```
npm start
```

## Features List
- Customizable characters
- Variety of voice options
- Easy to generate animations without needing API keys
